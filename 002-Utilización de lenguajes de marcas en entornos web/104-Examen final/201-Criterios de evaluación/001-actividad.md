Preguntas específicas del examen

Crea la plantilla HTML + CSS de un portafolio personal. Crea una estructura HTML, y dentro del body, header, main y footer

Dentro del footer, crea un article de muestra, con el contenido de cada pieza de portafolio: titulo, descripcion, fecha, categoria e imagen

Multiplica ese article para dar la sensacion de contener muchos artículos

Crea un estilo sencillo pero profesional para el ejercicio

Asegúrate de usar los componentes Flexbox y Grid de CSS para realizar la maquetación
